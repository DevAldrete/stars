# Stars

Make your terminal display a starred sky!

## Install with Go

Requires Go installed.

```sh
go install github.com/devaldrete/stars@latest
```

Make sure Go's binary directory is on your `PATH`:

```sh
export PATH="$(go env GOPATH)/bin:$PATH"
```

Then run:

```sh
stars
```

## Usage

```sh
stars
```

Keys:

- `q` quit
- `r` reseed stars
